Detailed Description

Class managing the user interface and corresponding actions.

Creates the different callbacks with material inputs, and links the relevant ones with corresponding actions (mesh rotating, render mode...). Allows for an easy update of the application parameters.

void CONTROLS::set_view ( GLFWwindow *  window)

Sets the camera according to the user choices.

Sets the camera position and orientation depending on the current mode: FLYING or STATIC Quaternions are used to rotate the view, and three main vectors are used:

  • cam defines the camera position
  • look defines the camera orientation
  • up defines the world's vertical
